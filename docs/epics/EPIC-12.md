# EPIC-12: Skills Enrichment (Optional)

> **Priority:** P3 (Could)  
> **Sprint:** Backlog  
> **Business Requirement:** BR-11  
> **Total Story Points:** 5

## Summary

Infer skills from project/role assignments and optionally import from external profile references. Enriches employee profiles without manual intervention.

## Dependencies

- **Upstream:** EPIC-01 (Workday data), EPIC-10 (privacy policy for external imports)
- **Downstream:** EPIC-04 (enriched profiles improve search results)

## Acceptance Criteria (Epic Level)

- Skills inferred from role assignments labeled as 'inferred' with source
- External profile import labeled and reviewable
- Policy compliance for external data import
