# US-041: Infer skills from project/role assignments

> **Epic:** [EPIC-12](../epics/EPIC-12.md) — Skills Enrichment (Optional)  
> **Priority:** P3 (Could)  
> **Story Points:** 3  
> **Sprint:** Backlog  
> **Status:** New  
> **Business Requirement:** BR-11

## Description

As a **Boss**, I want the system to infer skills from internal project/role assignments when available, so that employee profiles are enriched without manual intervention.

## Acceptance Criteria

- [ ] When inference logic is enabled and an employee has a role assignment (e.g., Java developer), the skill is added as 'inferred' with its source
