# US-049: Auto-convert talent preview to Resource Request on Stage 4B

> **Epic:** [EPIC-17](../epics/EPIC-17.md) — CPQ Pre-Sales Talent Preview  
> **Priority:** P1 (Must)  
> **Story Points:** 5  
> **Sprint:** Backlog  
> **Status:** New  
> **Business Requirement:** BR-CPQ

## Description

As a **Pre-Sales Lead**, I want the system to automatically convert an active talent preview into a Resource Request when the associated CPQ deal progresses to Stage 4B, so that pre-sales candidate selections carry forward into the staffing process without duplication.

## Acceptance Criteria

- [ ] When a CPQ opportunity linked to a talent preview reaches Stage 4B, the system creates a Resource Request populated with the selected talent preview candidates
- [ ] The account team is notified of the automatic conversion
- [ ] The original talent preview record remains accessible and is linked to the new Resource Request
