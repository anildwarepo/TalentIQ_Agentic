# US-017: Generate standardized CV in DXC format with configurable templates

> **Epic:** [EPIC-05](../epics/EPIC-05.md) — Resume / CV Generation  
> **Priority:** P1 (Must)  
> **Story Points:** 5  
> **Sprint:** Sprint 4  
> **Status:** New  
> **Business Requirement:** BR-03

## Description

As a **tender/tender contributor**, I want to generate a standardized CV in DXC format for a selected candidate, using a CV template that can be configured per offering or per client, so that CVs delivered for specific accounts or service lines follow the correct branding and structure requirements.

## Acceptance Criteria

- [ ] Clicking Generate Resume produces a resume document following the approved template structure
- [ ] After generation, I can download the resume document
- [ ] An admin screen allows uploading and activating multiple CV templates, each tagged with one or more offerings and/or client names
- [ ] When generating a CV, the system selects the template matching the active offering/client context; if no specific template matches, the default DXC template is used
- [ ] Template activation/deactivation is logged in the audit trail
