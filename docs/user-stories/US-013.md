# US-013: Export shortlist/search results to Excel, CSV, PDF or PPTX

> **Epic:** [EPIC-04](../epics/EPIC-04.md) — Candidate Search & CV Search  
> **Priority:** P1 (Must)  
> **Story Points:** 5  
> **Sprint:** Sprint 3  
> **Status:** New  
> **Business Requirement:** BR-01

## Description

As a **Manager/Account Lead**, I want to export the current shortlist/search results to Excel, CSV, PDF or PPTX, so that I can collaborate offline, deliver a client-ready document, or use the list in subsequent tendering/staffing processes.

## Acceptance Criteria

- [ ] Clicking Export generates a file with visible candidates and key attributes
- [ ] Export includes a summary of applied filters
- [ ] Available formats: .xlsx, .csv, .pdf and .pptx
- [ ] PDF and PPTX exports follow DXC branding and produce one page/slide per candidate with key attributes and CV summary
- [ ] The export is generated within 60 seconds for shortlists of up to 10 candidates
