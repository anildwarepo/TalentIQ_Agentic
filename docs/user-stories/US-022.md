# US-022: Package certification tests and bundled candidate profiles

> **Epic:** [EPIC-06](../epics/EPIC-06.md) — Certifications and Test Handling  
> **Priority:** P1 (Must)  
> **Story Points:** 5  
> **Sprint:** Sprint 4  
> **Status:** New  
> **Business Requirement:** BR-04

## Description

As a **tender/tender contributor / Account Lead**, I want to assemble a response package for a curated list of candidates that bundles their profiles together with certification tests, so that I can deliver a coherent set of profiles and supporting evidence to the client or internal stakeholders in a single artifact.

## Acceptance Criteria

- [ ] When multiple candidates are selected and a test package is requested, the system generates a consolidated set of evidence files for those candidates
- [ ] From any search or shortlist, candidates can be added to a named response package
- [ ] Each package records: opportunity name, creation date, creator, and the list of selected candidates with their key attributes
- [ ] Packages are saved and can be retrieved, edited, or duplicated
- [ ] A response package can be exported using US-013 (PDF/PPTX/Excel/CSV)
