# US-048: Auto-trigger talent preview on qualifying RFI/NSSR

> **Epic:** [EPIC-17](../epics/EPIC-17.md) — CPQ Pre-Sales Talent Preview  
> **Priority:** P1 (Must)  
> **Story Points:** 5  
> **Sprint:** Backlog  
> **Status:** New  
> **Business Requirement:** BR-CPQ

## Description

As a **Pre-Sales Lead**, I want the platform to automatically initiate a talent preview when a qualifying RFI or NSSR opportunity is detected in CPQ, so that account teams receive candidate proposals without waiting for a manual search.

## Acceptance Criteria

- [ ] When an opportunity is classified as RFI or NSSR, the system automatically initiates a talent preview session linked to that opportunity
- [ ] The account team is notified that a talent preview has been triggered
- [ ] The auto-triggered preview is traceable to the originating CPQ opportunity ID
