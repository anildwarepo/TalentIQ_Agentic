# US-023: Dashboard with certifications, skills, languages summary and RFI coverage heat map

> **Epic:** [EPIC-07](../epics/EPIC-07.md) — Manager Dashboards (People Analytics)  
> **Priority:** P1 (Must)  
> **Story Points:** 8  
> **Sprint:** Sprint 5  
> **Status:** New  
> **Business Requirement:** BR-05

## Description

As a **Manager / Pre-Sales Lead**, I want a dashboard that summarizes certifications, skills, and languages across the population (e.g., Iberia), and a heat map comparing the skills required by an RFI against the skills available in the bench, so I can understand capacity coverage and gaps and instantly see which RFI requirements are well-covered and which are at risk.

## Acceptance Criteria

- [ ] In the population panel, I can filter and see aggregated views by certifications/skills/languages
- [ ] Given an RFI with a defined skills/certification requirement list, the system generates a heat map where each cell represents a required skill and its coverage level across the bench (green = fully covered, amber = partially covered, red = not covered)
- [ ] The heat map updates in real time as candidates are added to or removed from the bench scope
- [ ] The heat map is exportable as part of the response package (US-022)
