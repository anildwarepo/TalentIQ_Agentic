# US-052: Talent preview analytics — response time and win rate impact

> **Epic:** [EPIC-17](../epics/EPIC-17.md) — CPQ Pre-Sales Talent Preview  
> **Priority:** P1 (Must)  
> **Story Points:** 5  
> **Sprint:** Backlog  
> **Status:** New  
> **Business Requirement:** BR-CPQ

## Description

As a **Pre-Sales Director**, I want dashboards showing talent preview response time and correlation with deal win rate, so that I can measure the business impact of the pre-sales talent process and identify improvement areas.

## Acceptance Criteria

- [ ] A talent preview analytics dashboard shows: average time from RFI detection to preview delivery, number of previews per period, and win rate for deals where a talent preview was provided vs not provided
