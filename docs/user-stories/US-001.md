# US-001: Candidate attributes from Workday

> **Epic:** [EPIC-01](../epics/EPIC-01.md) — Workday Data Integration  
> **Priority:** P1 (Must)  
> **Story Points:** 3  
> **Sprint:** Sprint 1  
> **Status:** New  
> **Business Requirement:** BR-02

## Description

As a **Manager/Account Lead**, I want the candidate's attributes (skills, certifications, languages, job level, location, service line, manager) to come from Workday, so that the application uses an authoritative dataset and reduces manual work.

## Acceptance Criteria

- [ ] Candidate profile in the app shows attributes derived from Workday
- [ ] Updated attributes are reflected in the app according to the agreed cadence

## Tags

`Workday` `EPIC-01` `Must` `TalentAI`

## Dependencies

- Workday API access with production credentials
- Agreed refresh cadence for attribute updates
