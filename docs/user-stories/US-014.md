# US-014: Identify skill gaps and suggest employees for retraining

> **Epic:** [EPIC-04](../epics/EPIC-04.md) — Candidate Search & CV Search  
> **Priority:** P1 (Must)  
> **Story Points:** 5  
> **Sprint:** Sprint 4  
> **Status:** New  
> **Business Requirement:** BR-01

## Description

As a **Manager/Account Lead**, I want the tool to explicitly identify skills required by a search/RFI that are not covered by any available bench candidate, and to suggest employees for retraining or training to close those gaps, so that I can proactively address capability shortfalls before submitting a response.

## Acceptance Criteria

- [ ] Each search result shows the option to suggest training to complete the employee's profile based on skills and certifications in the search
- [ ] After running a search, the system displays a gap list of skills/certifications for which zero qualified candidates exist
- [ ] Each gap entry shows the required skill, the number of candidates with partial match, and a suggested action (e.g. training, external hire)
- [ ] The gap list is exportable and feeds the skills coverage view (US-023)
