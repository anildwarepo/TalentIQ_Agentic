# US-037: Role-based access control for CV and certification data

> **Epic:** [EPIC-10](../epics/EPIC-10.md) — Privacy, Access Control and Compliance  
> **Priority:** P1 (Must)  
> **Story Points:** 3  
> **Sprint:** Sprint 1  
> **Status:** New  
> **Business Requirement:** BR-09

## Description

As a **Security/Privacy stakeholder**, I want role-based access control for CV content and certification evidence, so that sensitive employee data is only visible to authorized users.

## Acceptance Criteria

- [ ] When a user without permission tries to access CVs or tests, access is denied and logged
