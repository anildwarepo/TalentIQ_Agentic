# US-047: CPQ RFI/NSSR type detection

> **Epic:** [EPIC-17](../epics/EPIC-17.md) — CPQ Pre-Sales Talent Preview  
> **Priority:** P1 (Must)  
> **Story Points:** 3  
> **Sprint:** Backlog  
> **Status:** New  
> **Business Requirement:** BR-CPQ

## Description

As a **Pre-Sales Lead**, I want the system to automatically identify when a CPQ opportunity is of type RFI or NSSR, so that the platform can determine whether a talent preview is required without manual classification.

## Acceptance Criteria

- [ ] When a CPQ opportunity record is received, the system reads its type field and classifies it as RFI, NSSR, or other
- [ ] Only opportunities classified as RFI or NSSR trigger the pre-sales talent preview workflow
- [ ] The classification result is logged and visible in the opportunity detail view
