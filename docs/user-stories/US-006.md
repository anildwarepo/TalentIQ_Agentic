# US-006: Pre-load EQF/MECES mapping table from official sources

> **Epic:** [EPIC-03](../epics/EPIC-03.md) — EQF / MECES Study Mapping  
> **Priority:** P1 (Must)  
> **Story Points:** 3  
> **Sprint:** Sprint 2  
> **Status:** New  
> **Business Requirement:** BR-EQF

## Description

As a **system administrator**, I want the EQF/MECES mapping table to be pre-loaded automatically from the official frameworks on first setup, so that the platform starts with an up-to-date and authoritative qualification mapping without manual data entry.

## Acceptance Criteria

- [ ] Given the system is initialised, when the EQF/MECES pre-load process runs, all standard qualification levels (EQF 1–8 and their MECES equivalents for Spain) are imported and stored in the mapping table
- [ ] The pre-loaded entries include: qualification name, EQF level (1–8), MECES level (1–4), and the country/framework scope
- [ ] The import process logs the source, version, and date of the loaded data
- [ ] Pre-loaded entries are marked as 'Official' to distinguish them from administrator-added entries
