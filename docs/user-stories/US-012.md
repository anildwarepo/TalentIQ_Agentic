# US-012: Show triage attributes in search results

> **Epic:** [EPIC-04](../epics/EPIC-04.md) — Candidate Search & CV Search  
> **Priority:** P1 (Must)  
> **Story Points:** 2  
> **Sprint:** Sprint 3  
> **Status:** New

## Description

As a **Manager/Account Lead**, I want the search results to show the attributes necessary for quick triage (certification/supplier, location, service line, position level, EQF/MECES level, manager), so that I can pre-screen without opening all profiles.

## Acceptance Criteria

- [ ] Each result row includes the key attributes configured for triage, including EQF/MECES level
