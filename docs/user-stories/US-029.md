## Description

As a **Manager**, I want to trigger automatic reminder campaigns to update the resume, so that employees keep their resumes updated without manual HR campaigns.

## Acceptance Criteria

- [ ] When a CV freshness threshold is set and a CV refresh campaign is started, only employees exceeding that threshold are notified
