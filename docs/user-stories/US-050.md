# US-050: Pre-sales to staffing handoff

> **Epic:** [EPIC-17](../epics/EPIC-17.md) — CPQ Pre-Sales Talent Preview  
> **Priority:** P1 (Must)  
> **Story Points:** 3  
> **Sprint:** Backlog  
> **Status:** New  
> **Business Requirement:** BR-CPQ

## Description

As a **Staffing Manager**, I want to receive the candidate selections made during the pre-sales talent preview when a deal is handed off to staffing, so that I do not have to repeat the bench search from scratch.

## Acceptance Criteria

- [ ] When a talent preview is converted to a Resource Request (manually or automatically), all candidate selections and their preview metadata are included in the handoff record
- [ ] The staffing team can accept, replace, or add to the carried-forward candidates
- [ ] The handoff date, originating pre-sales owner, and CPQ opportunity ID are recorded on the Resource Request
